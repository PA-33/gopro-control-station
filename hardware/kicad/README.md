# KiCad project

Put the full KiCad source project here:

- `*.kicad_pro`
- `*.kicad_sch`
- `*.kicad_pcb`

Optional, only if you create local libraries later:

- `symbols/` for custom symbol libraries
- `footprints/` for custom footprint libraries

Use `exports/` for generated outputs only:

- Gerbers
- drill files
- PDF schematics
- BOM / pick-and-place files
